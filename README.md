# Cassidys Crystals

Cassidys Crystals is a starter crystal selling website focused on being super easy for customers and simple for admins.

## Purpose

- Sell all kinds of crystals and rare gems.
- Let customers request custom jewelry pieces.
- Give admins a clear place to manage inventory, orders, and custom requests.

## What is included

- Static customer storefront.
- Product category filters.
- Simple cart count interaction.
- Custom jewelry request form layout.
- Admin feature preview for inventory, orders, and request tracking.

## Open the site

Open `index.html` in a browser, or publish the repo with GitHub Pages.
